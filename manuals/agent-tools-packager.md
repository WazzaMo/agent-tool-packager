# Agent-Tools / Packager

A tool that can be used to create packages to install into many CLI Coding Agents.
It can be used to create packages and keep the packages in a local Station and used
in project Safehouses because any good Agent is relying on the Quartermaster to provide
good tools in the Station for use in the field, projects, and kept in a Safehouse for
ready use.
