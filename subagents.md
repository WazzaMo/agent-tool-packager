## Agent group rough plan

Agents can be in the foreground or background (async / immediate return)

Main agent - the control that human talks with.

### Built-in subagents

Explore - explores code and protects context window

Bash - runs commands

Browser - controls a browser via MCP tools.

### Custom subagents

Planner - divide larger tasks into small, focussed tasks.
